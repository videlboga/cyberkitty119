О платформе

Подключение к платформе MAX для партнёров и её сервисам — чат-ботам, мини-приложениям, каналам — доступно для юрлиц и ИП, которые являются резидентами РФ

Платформа MAX для партнёров позволяет выстраивать удобную коммуникацию с клиентами в мессенджере MAX

С помощью платформы вы можете создать чат-боты для настройки сценариев диалогов, подключить мини-приложения для обработки запросов и принятия заказов без перехода на внешние платформы, добавить канал своей организации в MAX для публикации новостей, а также подключить сервисы партнёров

Платформа MAX для партнёров пока доступна только в веб-версиях:

  • Для компьютера — работать лучше на экранах от 1440 px, ограничений на браузеры нет
  • Для мобильных устройств с iOS и Android — поддерживаются экраны от 375 px

О чат-ботах

Чат-боты экономят время

С помощью чат-ботов вы можете автоматизировать работу с клиентами, вести бизнес, обучать, продавать и развлекать

Бот создаётся после подключения к платформе MAX для партнёров, добавления там организации и её верификации. Бот проходит модерацию и после этого становится доступен пользователям через MAX

Для быстрого старта

Если у вас нет навыков программирования или нужен быстрый старт, используйте конструкторы сервисов партнёров, которые помогут собрать сценарии для бота без кода, по готовым шаблонам

Об интеграции с сервисами партнёров

После добавления и модерации бота на платформе вы можете подключить сервисы партнёров:

  • Конструкторы ботов — для создания сценариев чат-ботов без разработки
  • CRM-системы — для автоматизации рутинных операций и работы с клиентской базой: лидами, заявками, историей общения с клиентами
  • Чат-платформы — для общения с клиентами через сервис партнёров

Для подключения к сервисам партнёров используйте токен бота

О мини-приложениях

Мини-приложения

Мини-приложения — это веб-сайты и программы, которые можно запустить прямо в MAX: например, интернет-магазины или игры

Мини-приложения снижают нагрузку на команды продаж и поддержки за счёт возможности принятия заказов без перехода на внешние ресурсы

Чтобы создать мини-приложение, нужны навыки программирования

Мини-приложения доступны на разных устройствах и проще в разработке, чем мобильные приложения. Они работают на стандартных веб-технологиях — HTML, JavaScript, CSS, а ещё их не нужно дорабатывать под разные платформы и модерировать в сторах

Мини-приложения не могут существовать изолированно

Для возможности взаимодействовия мини-приложений с API подключите библиотеку MAX Bridge. Это посредник на стороне пользователя, с помощью которого приложение обменивается данными с клиентской частью MAX. Библиотека нужна для отправки и получения событий. Подробнее читайте в разделе «MAX Bridge»

Чтобы пользователи не замечали переходы из чатов в мини-приложения, рекомендуем адаптировать интерфейсы сервисов под дизайн-систему MAX. Создавайте консистентный и привлекательный визуал с библиотекой React-компонентов MAX UI

Мини-приложения в MAX должны соответствовать правилам

Перед размещением мини-приложения в MAX изучите Правила платформы. Ваш сервис должен соответствовать законам Российской Федерации, а также требованиям к его содержанию и функциональности

О каналах

В МАХ доступны публичные и приватные каналы

С помощью публичных каналов вы можете взаимодействовать со своей аудиторией через мессенджер MAX, например публиковать новости и анонсировать акции

Приватные каналы удобно использовать для коммуникаций на ограниченную аудиторию, например, оповещений и публикации новостей только для сотрудников организации. Количество приватных каналов неограниченно — можно создать несколько, например, для разных отделов или филиалов

Также вы можете использовать приватные каналы для монетизации. Например, публиковать уникальный контент и выдавать к нему доступ только после оплаты или размещать рекламные посты за деньги

Метка А+ присваивается ресурсам на платформах ВКонтакте, Одноклассники, Дзен и Telegram. Чтобы проверить, есть ли у вашего сообщества или канала метка А+, откройте бота Каналы в МАХ — он автоматически покажет доступные ресурсы с меткой

Если у вас уже есть ресурсы с меткой А+ на других платформах, вы можете создать публичные каналы с такими же никами и названиями в MAX

Для этого воспользуйтесь ботом Каналы в МАХ. Можно добавить столько каналов, сколько у вас есть ресурсов с меткой А+ на платформах ВКонтакте, Дзен, Одноклассники или в Telegram

Обратите внимание, что создавать канал в MAX нужно с того же номера, с которого вы подавали заявку на получение метки А+ для стороннего ресурса

Создать канал с названием и ником, как у ресурса с меткой А+ на другой платформе

Также вы можете создать публичный канал с выбранным вами названием и автогенерируемым ником

Создать такой канал можно независимо от наличия ресурсов с меткой А+ и регистрации в Роскомнадзоре. Пока доступно создание только одного такого канала. Вам потребуется войти на платформу или зарегистрироваться, создать и верифицировать организацию

После этого вы сможете добавить канал через бота Каналы в MAX для бизнеса. Название канала вы выбираете сами, а ник генерируется автоматически. Обратите внимание, что профиль в MAX должен быть создан на тот же номер телефона, что вы использовали при регистрации на платформе

Для безопасности мы разделили добавление и повседневное использование канала между двумя платформами:

  • В MAX для партнёров вы верифицируете организацию и инициируете создание канала
  • В мессенджере MAX вы непосредственно создаёте канал, управляете его настройками и публикуете посты

Приватный канал можно создать прямо в мессенджере МАХ

Для создания приватного канала подключаться к платформе МАХ для партнёров не нужно — достаточно быть пользователем мессенджера МАХ с аккаунтом, зарегистрированным на российский номер телефона. Рекомендуем использовать телефон организации или ответственного, кто будет управлять каналом и модерировать заявки на вступление

Для дополнительной безопасности вы можете включить в приватных каналах опцию Заявка на вступление: по умолчанию она отключена. В этом случае пользователи смогут подписаться на канал только после согласования заявки администратором
