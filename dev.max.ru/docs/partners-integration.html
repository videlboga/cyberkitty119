Об интеграции с сервисами партнёров

Подключение к платформе MAX для партнёров и её сервисам — чат-ботам, мини-приложениям, каналам — доступно для юрлиц и ИП, которые являются резидентами РФ

После добавления и модерации бота вам станет доступен токен бота, с помощью которого вы можете подключить сервисы партнёров:

  • Чат-платформы поддержки — для общения с клиентами через сервис партнёров
  • Конструкторы ботов – для создания сценариев чат-ботов без разработки
  • CRM-системы — для автоматизации рутинных операций и работы с клиентской базой: лидами, заявками, историей общения с клиентами

Токен бота для интеграции

Для подключения к сервисам партнёров используйте токен бота. Его можно найти на платформе в разделе Чат-боты → Интеграция → Получить токен — токен находится в поле с одноимённым названием

Интеграция с искусственным интеллектом помогает автоматизировать рутинные задачи и снижает нагрузку на персонал

CRM-системы

CRM-системы помогают автоматизировать бизнес-процессы

Входящие запросы из разных каналов связи объединяются в единую систему и распределяются между сотрудниками автоматически, что позволяет ускорить обработку заявок. CRM-системы позволяют управлять воронками продаж, обрабатывать уведомления, генерировать отчёты и аналитику, настраивать сценарии и триггеры. Также современные CRM сертифицированы государственными органами и обеспечивают высокий уровень защиты конфиденциальных данных

Вы можете подключить созданного в MAX бота к CRM-системам — ознакомьтесь со списком рекомендуемых партнёров ниже. После подключения все сообщения из бота будут автоматически попадать в CRM
