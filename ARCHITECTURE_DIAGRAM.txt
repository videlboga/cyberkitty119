БЫЛО (Синхронная обработка):
    User sends file
        │
        ├─ Bot downloads (30 sec)
        ├─ Bot extracts audio (10 sec)
        ├─ Bot compresses (5 sec)
        ├─ Bot TRANSCRIBES (5-30 MIN) ◄─────── 🔴 БОТ ЗАМОРОЖЕН!
        ├─ Bot formats (10 sec)
        ├─ Bot creates note (5 sec)
        ├─ Bot sends result (5 sec)
        │
    ▼ Total: 6-40 MINUTES (bot blocked!)
    User: 😫 (stuck, bot not responding)


СТАЛО (Асинхронная обработка):
    User sends file
        │
        ├─ Bot downloads (30 sec)
        ├─ Bot extracts audio (10 sec)
        ├─ Bot compresses (5 sec)
        ├─ Bot enqueues (1 sec) ◄─────────── ⚡ БОТ СВОБОДЕН!
        │
    ▼ Total: ~1 MINUTE (bot responds immediately!)
    User: "✅ Accepted! Processing..." ← видит сразу!
        │
        └─ Worker processes in background
            ├─ Downloads (already have audio)
            ├─ TRANSCRIBES (5-30 MIN) ✅ (не блокирует бот!)
            ├─ Finalizes (10 sec)
            └─ Delivers (5 sec)

РЕЗУЛЬТАТ: Bot can handle 10-100x more users! 📈
