PERFORMANCE ANALYSIS
================================================================================

WHY 3.50x SPEEDUP FOR 5 TASKS (NOT 5.0x)?

1. GPU ARCHITECTURAL LIMITS
   - RTX 3070 Ti has limited parallel execution units
   - Can schedule 5-6 concurrent CUDA streams, but not in perfect parallel
   - Each stream competes for SM (Streaming Multiprocessor) resources
   - Memory bandwidth is shared among processes

2. MEMORY BEHAVIOR
   - Peak VRAM usage: 3.49GB (45.6%)
   - Memory is NOT the bottleneck
   - 4.2GB remaining unused during peak load
   - Could theoretically fit 8-10 models, but:
     * Cache corruption when >5 processes load simultaneously
     * Model deserialization thread safety issue

3. COMPUTATIONAL LIMITS
   - NVIDIA's GPU scheduler serializes some operations
   - Attention layers (core of Whisper) have shared resource contention
   - GPU utilization per process decreases with more parallel tasks

4. ✓ Each task gets adequate GPU bandwidth

ARCHITECTURAL INSIGHTS
================================================================================

The GPU parallelism is LIMITED BY:
  1. Model cache loading (file system I/O bottleneck)
  2. CUDA kernel serialization (not purely parallel operations)
  3. GPU memory bandwidth sharing (not VRAM capacity)
  4. NVIDIA scheduler prioritization

The GPU is NOT limited by:
  1. VRAM capacity (7.7GB vs 3.49GB actual peak)
  2. Memory bandwidth (theoretical limit not reached)
  3. Computational power (only ~45% utilized at peak)
